## 불사의 의미

> "인간이 불사를 획득하는 유일한 방법은 낳는 것이다.
> 몸으로 낳는 것은 자식이고, 영혼으로 낳는 것은 지혜와 덕이다.
> 호메로스와 헤시오도스의 시가 불사인 것은
> 그것이 세대를 넘어 살아남기 때문이다." — 향연 208e-209e

## 보존 절차

### Step 1: eros-analysis.md 생성

```
Write .claude/docs/eros-analysis-{YYYYMMDD}-{주제요약}.md
```

- 파일명에 날짜와 주제 요약 포함
- `.claude/docs/` 디렉토리에 저장

### Step 2: learnings.md 업데이트

Phase 4-5에서 도출된 학습을 `.claude/memory/learnings.md`에 추가:

```markdown
## {날짜} - {주제} (via /eros)

- Layer 5 원리: {불변 원리}
- 메타 패턴: {Layer 4 메타 학습}
- 적용 범위: {어떤 상황에서 이 학습이 유용한가}
```

주의: learnings.md가 없으면 새로 생성. 있으면 하단에 추가.

### Step 3: decisions.md 업데이트 (ADR 산출 시)

Phase 5에서 ADR이 산출되었으면 `.claude/memory/decisions.md`에 추가.
