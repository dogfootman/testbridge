## 6. 발견된 이슈 및 해결

### 6.1 수정된 이슈 (16개)

#### Notifications API (9개)
**문제**: userId 쿼리 파라미터 누락으로 400 에러
**해결**: getSession()으로 userId 추출
**영향**: Notifications 테스트 통과율 80% → 95%

#### Categories API (6개)
**문제**: resetCategoryCache 함수 미export
**해결**: export function 추가
**영향**: Categories 테스트 100% 통과

#### Sidebar Component (1개)
**문제**: 중복 요소로 쿼리 실패
**해결**: getAllByLabelText 사용
**영향**: Sidebar 테스트 100% 통과

### 6.2 남은 이슈 (7개)

#### Tester Home 검색 (4개)
**문제**: Debounce 타이밍 이슈
**우선순위**: 낮음
**해결 방안**: waitFor 타임아웃 증가 또는 Debounce 시간 조정

#### Feedback Form 버그 리포트 (3개)
**문제**: 비동기 처리 타이밍
**우선순위**: 중간
**해결 방안**: Promise.all 또는 순차 처리 개선
