## 1. 제품 개요

### 1.1 서비스 소개

**서비스명**: TestBridge
**한줄 소개**: 안드로이드 개인 개발자와 테스터를 연결하여 Google Play 비공개 테스트 요건(14일 / 14명)을 충족시켜주는 매칭 플랫폼
**플랫폼**: 웹사이트 (반응형) + 모바일 앱 (2차)

### 1.2 핵심 문제 정의

Google Play는 2023년 11월부터 신규 개인 개발자 계정에 대해 프로덕션 배포 전 **비공개 테스트 트랙에서 20명 중 14명 이상이 14일 연속 옵트인**을 유지하도록 요구하고 있습니다.

**개발자의 고충**:
- 주변에 20명의 테스터를 구하기 어려움
- 14일간 지속적으로 참여시키기 더 어려움
- 커뮤니티에 테스터 모집 글을 올려도 반응이 저조
- 테스터 이탈 시 다시 처음부터 카운트

**테스터의 니즈**:
- 새로운 앱을 남들보다 먼저 체험하고 싶은 욕구
- 부수입 또는 리워드를 받을 수 있는 기회
- 앱 개발 생태계에 기여하고 싶은 마음

### 1.3 해결책

TestBridge는 개발자와 테스터를 연결하여 양쪽의 니즈를 해결합니다:

1. **개발자**: 14일 테스트 요건을 쉽게 충족 + 고품질 피드백 수집
2. **테스터**: 신규 앱 조기 체험 + 리워드 수령
3. **플랫폼**: 신뢰도 기반 매칭으로 이탈 최소화

## 3. 리워드 시스템

### 3.1 리워드 구조 (하이브리드)

TestBridge는 **유료 리워드**와 **상호 테스트 크레딧** 두 가지 모드를 지원합니다.

#### Option A: 유료 리워드 (포인트 기반)
- 개발자가 1인당 리워드 금액 설정 (1,000~8,000원)
- 테스트 완료 시 포인트 지급
- 포인트 → 현금 출금 (최소 10,000원) 또는 기프티콘 교환

#### Option B: 상호 테스트 (크레딧 교환)
- 다른 개발자 앱 테스트 완료 시 크레딧 적립 (1앱당 20크레딧)
- 크레딧으로 자신의 앱 등록 (50크레딧)
- 무료로 운영 가능, 개발자 커뮤니티 형성

### 3.2 리워드 유형별 금액 가이드라인

| 테스트 유형 | 권장 리워드 (1인당) |
|------------|-------------------|
| 기본 (14일 옵트인만) | 1,000 ~ 3,000원 |
| 피드백 포함 | 3,000 ~ 5,000원 |
| 심화 (버그 리포트 + 상세 리뷰) | 5,000 ~ 8,000원 |

## 4. 수익 모델

### Phase 1: 초기 (사용자 확보 단계)
- **무료 기본 플랜**: 월 1개 앱 등록, 기본 대시보드
- **건별 결제**:
  - 20명 모집: 29,000원
  - 30명 모집: 39,000원
- **플랫폼 수수료**: 리워드 지급 시 10~15%

### Phase 2: 성장기
- **프리미엄 구독**:
  - Basic: 월 19,900원 (무제한 앱 등록, 수수료 10→5%)
  - Pro: 월 49,900원 (우선 매칭, 상세 분석 리포트, 수수료 5→0%)
  - Enterprise: 월 99,900원 (B2B, API, 전담 지원)
- **긴급 모집 부스트**: 1회 5,000원 (상단 노출)

### Phase 3: 확장기
- **광고**: 앱 탐색 페이지 내 개발 도구/서비스 광고
- **B2B 서비스**: 기업 개발팀 대상 대량 테스트 관리 솔루션

## 5. 차별화 포인트

### 5.1 신뢰도 시스템

테스터의 이탈은 개발자에게 치명적입니다. 신뢰도 시스템으로 이를 방지합니다.

- **신뢰도 점수**: 테스트 완료율(40%), 이탈 횟수(30%), 피드백 품질(20%), 활동 기간(10%) 기반 0~100점 자동 산정
- **배지 시스템**: 브론즈(0~49) / 실버(50~69) / 골드(70~89) / 다이아몬드(90~100)
- **혜택 차등**: 높은 등급 테스터는 더 높은 리워드의 테스트에 우선 지원 가능
- **페널티**: 중도 이탈 시 신뢰도 -10점, 반복 이탈 시 일정 기간 지원 제한

### 5.2 테스트 보험

- 테스터 이탈 시 자동으로 대기 테스터에게 알림 발송 (D+10 이전)
- 개발자는 여유 인원(예: 25명) 선정 가능 (최소 20명 충족 보장)

### 5.3 자동화

- Google Play Console 연동 (가능한 범위 내)으로 옵트인 상태 자동 추적
- 14일 완료 시 자동 리워드 지급 옵션
