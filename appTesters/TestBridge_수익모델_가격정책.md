# TestBridge 수익모델 & 가격 정책

## 1. 수익 구조 개요

```
수익원 = 구독료 + 건별 이용료 + 테스터 리워드 수수료
```

- **과금 대상**: 개발자만 (테스터는 무료)
- **통화**: 원화(KRW) + 달러(USD) 병행
- **구독료와 리워드는 별도**: 구독 = 플랫폼 이용료 / 리워드 = 테스터에게 지급하는 비용

---

## 2. 결제 방식: 건별 vs 구독

개발자는 자신의 상황에 맞게 선택할 수 있습니다.

| 방식 | 대상 | 특징 |
|------|------|------|
| 건별 결제 (Pay-per-use) | 앱 1~2개만 출시하는 개발자 | 필요할 때만 결제, 진입장벽 낮음 |
| 구독 (Subscription) | 앱을 지속적으로 출시하는 개발자/스튜디오 | 월 정액제, 건당 단가 할인 |

---

## 3. 건별 결제 (Pay-per-use)

구독 없이 1건씩 이용할 수 있는 옵션입니다.

| 항목 | 가격 (KRW) | 가격 (USD) |
|------|-----------|-----------|
| 앱 1건 등록 (테스터 20명 매칭) | 29,000원 | $19.99 |
| 앱 1건 등록 (테스터 30명 매칭) | 39,000원 | $29.99 |
| 긴급 모집 부스트 (상단 노출) | 5,000원 | $3.99 |

- 테스터 리워드는 별도 지급
- 리워드 수수료: 리워드 금액의 15%

**예시) 건별로 앱 1건 등록 시 총 비용:**
```
앱 등록비:        29,000원
테스터 리워드:    20명 × 2,000원 = 40,000원
리워드 수수료:    40,000원 × 15% = 6,000원
──────────────────────────
총 비용:          75,000원 (약 $52)
```

### 각 플랜 상세

#### Free (무료)
- **대상**: 처음 앱을 출시하는 입문 개발자
- **핵심**: 상호 테스트(크레딧)로 무료 이용 가능
- **제한**: 월 1개 앱, 리워드 수수료 15%
- **목적**: 사용자 유입 → 유료 전환 유도

#### Basic (월 19,900원 / $14.99)
- **대상**: 개인 개발자, 사이드 프로젝트 다수 운영
- **핵심**: 월 10개 앱 등록, 수수료 10%로 할인
- **킬러 기능**: 긴급 부스트 월 1회 무료, 프로덕션 등록 PDF 가이드

#### Pro (월 49,900원 / $34.99)
- **대상**: 전업 개발자, 소규모 스튜디오
- **핵심**: 월 50개 앱, 신뢰도 높은 테스터 우선 매칭
- **킬러 기능**: 상세 피드백 리포트, 1:1 프로덕션 등록 가이드, 긴급 부스트 월 5회

#### Enterprise (월 99,900원 / $69.99)
- **대상**: 앱 개발 에이전시, 대량 출시 스튜디오
- **핵심**: 월 100개 앱, 최저 수수료 5%, 전담 매니저
- **킬러 기능**: API 제공 (자체 시스템 연동), 분석 리포트, 무제한 부스트

## 5. 테스터 리워드 구조

### 5.1 리워드 설정 (개발자)

개발자가 테스트 등록 시 리워드 금액을 직접 설정합니다.

| 리워드 유형 | 설명 | 권장 금액 (1인당) |
|------------|------|-----------------|
| 기본 리워드 | 14일 옵트인 유지만 | 1,000 ~ 2,000원 ($1~2) |
| 피드백 리워드 | 옵트인 + 피드백 작성 | 2,000 ~ 4,000원 ($2~3) |
| 심화 리워드 | 옵트인 + 상세 리뷰 + 버그 리포트 | 4,000 ~ 8,000원 ($3~6) |

- 개발자가 최소 리워드 이상으로 자유롭게 설정 가능
- 리워드가 높을수록 테스터 지원율 상승 → 빠른 매칭

### 5.2 리워드 수수료

플랫폼은 리워드 지급 시 수수료를 수취합니다.

| 플랜 | 수수료율 | 예시 (20명 × 2,000원) |
|------|---------|---------------------|
| Free / 건별 | 15% | 40,000원 + 6,000원 = 46,000원 |
| Basic | 10% | 40,000원 + 4,000원 = 44,000원 |
| Pro | 7% | 40,000원 + 2,800원 = 42,800원 |
| Enterprise | 5% | 40,000원 + 2,000원 = 42,000원 |

### 5.3 리워드 지급 조건

테스터는 아래 조건을 충족해야 리워드를 수령할 수 있습니다.

```
필수 조건:
✅ 14일 연속 옵트인 유지
✅ 앱 설치 및 최소 1회 실행

추가 조건 (개발자가 선택):
☐ 피드백 양식 작성
☐ 버그 리포트 제출
☐ 상세 리뷰 작성
```

### 5.4 테스터 리워드 수령 방식

| 방식 | 최소 출금 | 설명 |
|------|----------|------|
| 포인트 적립 | - | 플랫폼 내 포인트로 적립 |
| 현금 출금 | 10,000원 ($10) | 계좌이체 (국내) / PayPal (해외) |
| 기프티콘 교환 | 5,000원 ($5) | 카카오 기프티콘, 아마존 기프트카드 등 |

## 6. 상호 테스트 (크레딧 시스템)

리워드 비용을 아끼고 싶은 개발자를 위한 무료 옵션입니다.

### 크레딧 획득 방법
| 활동 | 크레딧 |
|------|--------|
| 다른 앱 테스트 완료 (14일) | +30 |
| 피드백 작성 | +10 |
| 버그 리포트 제출 | +15 |
| 신규 회원 추천 | +20 |

### 크레딧 사용
| 활동 | 크레딧 |
|------|--------|
| 내 앱 테스트 등록 | -50 |
| 긴급 모집 부스트 | -30 |

**상호 테스트 vs 유료 리워드 비교:**
```
상호 테스트: 개발자 시간 투자 → 무료 → 매칭 속도 느림 (2~7일)
유료 리워드: 개발자 비용 투자 → 유료 → 매칭 속도 빠름 (24~48시간)
```

## 8. 경쟁사 대비 가격 포지셔닝

```
저가 ◄─────────────────────────────────► 고가

Testers Community ($0~15)
    ▲ 우리(Free/건별)
         ▲ 20testers ($20~30)
              ▲ 우리(Basic $15)
                   ▲ 우리(Pro $35)
                        ▲ TesterMob
                             ▲ Beta Family (예산+10%)
                                  ▲ BetaTesting (기업가)
                                       ▲ 우리(Enterprise $70)
```

**포지셔닝 전략:**
- Free/건별: Testers Community와 동일 가격대에서 경쟁 (한국 시장 + 품질로 차별화)
- Basic~Pro: 중간 가격대에서 "가성비 좋은 프리미엄" 포지션
- Enterprise: 대량 이용 시 Beta Family/BetaTesting 대비 저렴

```
앱 1~2개만 출시?
├─ 비용 쓰기 싫다 → Free (상호 테스트)
├─ 빠르게 하고 싶다 → 건별 결제 (29,000원~)
│
앱을 지속적으로 출시?
├─ 월 10개 이하 → Basic (19,900원/월)
├─ 월 50개 이하 → Pro (49,900원/월)
├─ 월 100개 이하 → Enterprise (99,900원/월)
```
